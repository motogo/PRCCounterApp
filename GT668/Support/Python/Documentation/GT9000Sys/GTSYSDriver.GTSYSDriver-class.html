set_reference(self, reference)

source code 

Select reference clock option.

Parameters:
  • reference (int) - see GTSysReference
    • GTSYS_REF_INT - System internal timebase
    • GTSYS_REF_EXT - System external reference (connector in the back)
    • GTSYS_REF_AUTO - If 5MHz or 10Mhz signal is detected on the external connector - select external, otherwise - internal
    • GTSYS_REF_GPS - Use the available GPS (the function GTSysGetStatus can tell which, if any, GPS is installed).
Returns: bool
Returns False if setting failed, True if setting was successful.

set_arm_source(self, arm_source)

source code 

Select source of arming to be distributed.

Parameters:
  • arm_source (int) - see GTSysArmSource
    • GTSYS_ARM_SYS - Use Arming from the master site
    • GTSYS_ARM_EXT - Use external arming (connector in the back)
    • GTSYS_ARM_1PPS - Use 1PPS arming. The 1PPS will come from the GPS - if it's used, or will be generated from whatever reference is used.
Returns: bool
Returns False if setting failed, True if setting was successful.

set_arm_threshold(self, perc, val)

source code 

Set ARM input threshold (ignored if arming selection is not GTSYS_ARM_EXT).

Parameters:
  • perc (bool) - If True, the val is in percents, else the val is in Volts.
  • val (Double) - value of the input threshold (interpretation depends on the perc value).
Returns: bool
Returns False if setting failed, True if setting was successful.

NOTE: The external arming signal should be connected and active when this function is called with perc equal True (so that the system can measure the signal peaks and calculate the threshold voltage)

get_arm_threshold(self)

source code 

Get current setting of ARM input threshold. If the reading was successful, the function returns an Object with 'perc' : <Bool>, 'value': <Double> (see set_arm_threshold for valid values), else it returns None.

Returns:
If the reading was successful, the function returns an Object, else it returns None.

get_arm_actual_threshold(self)

source code 

Get actual ARM input threshold voltage. When GTSysSetArmThreshold is called with perc equal GT_True this function can be used to find what voltage was used.

Returns: None / double
