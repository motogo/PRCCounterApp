Class RawTimetagsSet

source code

Object used to retreive time tags from the read_raw method, contains three fields:

  • tags - int[], Buffer for raw data (32-bit for each time tag)
  • count - int, Number of words actually read
  • size - int, expected number of tags for channel 0

get_tags(self, start=0, end=None, step=1)

source code 

Retrieves tags. Allows defining a starting index, step, and end index. The returned lists will be sublists of the originals created based on the parameters. When none of the additional parameters (start, end, step) are specified, the method returns the whole lists.

Parameters:
  • start - starting index - index of the first element in the sublists.
  • end - ending index - index of the last element in the sublists.
  • step - determines the index step between the tags included in the sublists.
Returns: list
Sublist (or whole list) of the original time tags list.
