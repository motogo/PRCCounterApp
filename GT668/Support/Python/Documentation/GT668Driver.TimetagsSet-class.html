Class TimetagsSet

source code

Object used to retreive time tags from the read_timetags method, contains six fields:

  • channel0Tags - double[], Array with time tags from channel 0
  • channel0Count - int, Number of time tags actually read from channel 0
  • channel0Size - int, expected number of tags for channel 0
  • channel1Tags - double[], Array with time tags from channel 1
  • channel1Count - int, Number of time tags actually read from channel 1
  • channel1Size - int, expected number of tags for channel 1

get_tags(self, channel, start=0, end=None, step=1)

source code 

Retrieves tags for a given channel. Allows defining a starting index, step, and end index. The returned list will be a sublist of the original created based on the parameters. When none of the additional parameters (start, end, step) are specified, the method returns the whole list.

Parameters:
  • channel - channel index number.
  • start - starting index - index of the first element in the sublist.
  • end - ending index - index of the last element in the sublist.
  • step - determines the index step between the tags included in the sublist.
Returns: list
Sublist (or whole list) of the original time tags list.
